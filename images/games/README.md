# Game Images Folder

## How to Use

Place your game images in this folder with the following naming convention:

### Image Naming Format
Name your image files using the game ID from the games database:
- `game-1.jpg` for Elden Ring
- `game-2.jpg` for Baldur's Gate 3
- `game-3.jpg` for Cyberpunk 2077
- etc.

### Supported Formats
- `.jpg` or `.jpeg` (recommended)
- `.png`
- `.webp`

### Recommended Image Specifications
- **Aspect Ratio**: 16:9 (recommended)
- **Resolution**: 1280x720px or higher
- **File Size**: Under 500KB for optimal loading
- **Format**: JPG with compression for best balance

## Example
If you want to add an image for Elden Ring (ID: 1), name it `game-1.jpg` and place it in this folder.

The website will automatically detect and use your images!

## Fallback
If an image is not found, the website will display a colored gradient background with a category icon as a fallback.
